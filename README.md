# ロボットシステム学　課題２  
ROSからウェブに数値を飛ばしブラウザで閲覧する  
## 概要  
countのノードで数値をカウントし値をパブリッシュする  
カウントされた値をtwiceのノードでサブスクライブし値を２倍にしてパブリッシュする
簡易ネットサーバーを立ち上げる  
webserverのノードでtwiceの値をサブスクライブしブラウザで閲覧できるようにする  
## 環境  
CPU Core i5-6200U  
Ubuntu 16.04  
ROS kinetic
